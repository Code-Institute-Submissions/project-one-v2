Overview

Wembley Park, the home of Football, in undergoing a transformation. The land surrounding the iconomic stadium is becoming a new home for London and a retail and cultural destination for its visitors.

This website will provides Benchmark information in the on-going development of the Park. Currently there are three Projects which have either been completed or are under-going construction.

This Overview page combines key metrics on programme and cost from each of the projects, to enable users to see metrics side-by-side. Each project page provides further breakdown of these metrics, as well as a Gallery of project images.
